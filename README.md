# Squeeze Project Shopify Theme

Custom Shopify theme for [The Squeeze Project](https://thesqueezeproject.com) storefront — selling custom camp squeeze toys to fund camp scholarships via [SCOPE](https://scopeusa.org).

Based on [Shopify Horizon](https://github.com/Shopify/horizon), customized for The Squeeze Project.

## Connected to

- **Store:** https://squeeze-project.myshopify.com (password-protected during setup)
- **Fundraiser site repo:** https://github.com/thesqueezeproject/squeeze-project-form
- **Charity partner:** https://scopeusa.org

## Development

This repo is connected to the Shopify store via GitHub integration. Any commit to `main` auto-syncs to the connected theme in Shopify admin — no separate deploy step.

The theme is based on Shopify's Horizon. See [release-notes.md](release-notes.md) for upstream Horizon changes inherited from the base theme.
